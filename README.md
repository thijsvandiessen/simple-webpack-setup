# Simple webpack setup

setup: npm install

```
dev:    npm start
build:  npm run build
```

## Features

Handy features:

- Compiles scss to css
- A css autoprefixer
- Minify css
- Minify js
- A handy static folder
- Manifest
- Uses the latest JavaScript syntax
- And [dynamic import](https://babeljs.io/docs/en/babel-plugin-syntax-dynamic-import)
